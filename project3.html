Création d'un sudoku

Pour la SAE 1.01, j'ai crée un sudoku en C qui se joue à partir de l'interface de l'IDE.

Les Etapes


  • 1ere Etape
    J'ai commencé par la création d'une table 9X9.

  • 2eme Etape
    Ensuite j'ai importé les fichiers qui comporte les grilles initiales.

  • 3eme Etape
    Après l'importation des fichiers, j'ai implémenter les données des fichiers dans la table.

  • 4eme Etape
    Pour continuer ce projet,j'ai rajouter les contrôles pour que l'utilisateur puisse rajouter des valeurs dans la table.

  • 5eme Etape
    Pour finir le projet, j'ai ajouter des fonctions pour verifier si la valeur peut être insérer dans la table sans être deux fois sur la colonne, ligne ou diagonale.

Compétences utilisées lors de ce projet


  • Compétence 1 : développer une application simple
    Durant le projet, j'ai développé une application(Sodoku) simple en C.

  • Compétence 2 : Optimiser une application
    J'ai dû optimiser certaines méthodes pour une expérience plus agréable pour le joueur.
