SAE 1.04

Pour la SAE 1.04, nous devions créer un schéma SQL à partir d'un texte. Cette SAE est répartie en 2 parties, la première est la création du diagramme UML à partir du texte. La seconde est l'utilisation du diagramme UML pour faire la création du schéma.

Les Etapes


  • 1ere Etape
    Nous avons commencé par la création UML à partir d'un texte

  • 2eme Etape
    Ensuite nous avons dû utiliser le diagramme UML pour faire la création du schéma.


Compétences utilisées lors de ce projet


  • Compétence 4 : Gérer des données
    Nous avons du utiliser des données d'un diagramme UML créer par nous pour ensuite créer des tables SQL

  • Compétence 5 : Conduire un projet
    J'ai dirigé l'équipe lors de la création des Tables,et du diagramme UML.
